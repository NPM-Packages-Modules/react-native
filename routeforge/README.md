# routeforge

**React Native** library.

Generate complete Express CRUD modules automatically.

## Features

- route generation
- controller generation
- validation setup
- Swagger docs
- middleware injection
- folder scaffolding

## Example

```ts
import { routeforge } from "@mr-aftab-ahmad-khan/routeforge";

await routeforge.create("products");
```

## Why it matters

Backend teams repeatedly rebuild identical CRUD structures.
