# mongoforge

**React Native** library.

Automatically optimize MongoDB indexing strategies.

## Features

- slow query analysis
- index generation
- duplicate cleanup
- query profiling
- performance scoring
- index recommendations

## Example

```ts
import { mongoforge } from "@mr-aftab-ahmad-khan/mongoforge";

await mongoforge.optimize();
```

## Why it matters

Poor Mongo indexing destroys app performance silently.
