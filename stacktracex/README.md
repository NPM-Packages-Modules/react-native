# stacktracex

**React Native** library.

Visual distributed tracing for MERN applications.

## Features

- request timelines
- trace correlation
- latency visualization
- service mapping
- error tracing
- request replay

## Example

```ts
import { stacktracex } from "@mr-aftab-ahmad-khan/stacktracex";

stacktracex.track(app);
```

## Why it matters

Debugging distributed systems is incredibly painful.
