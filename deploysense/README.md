# deploysense

**React Native** library.

Detect risky deployments before production release.

## Features

- dependency checks
- migration validation
- env verification
- rollback alerts
- memory forecasting
- deployment scoring

## Example

```ts
import { deploysense } from "@mr-aftab-ahmad-khan/deploysense";

await deploysense.verify();
```

## Why it matters

Many production outages happen during deployment.
