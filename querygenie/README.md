# querygenie

**React Native** library.

Generate advanced MongoDB queries automatically.

## Features

- filtering
- search
- pagination
- sorting
- aggregation pipelines
- query optimization

## Example

```ts
import { querygenie } from "@mr-aftab-ahmad-khan/querygenie";

const q = querygenie(Product).filter(req.query);
```

## Why it matters

Developers constantly rebuild repetitive query systems.
