# apiflowx

**React Native** library.

Visualize and orchestrate complex API architectures.

## Features

- dependency graphs
- API mapping
- request replay
- latency visualization
- service flow analysis
- endpoint discovery

## Example

```ts
import { apiflowx } from "@mr-aftab-ahmad-khan/apiflowx";

apiflowx.inspect(app);
```

## Why it matters

Teams lose visibility as backend systems scale.
