# apidocsmith

**Topics:** `api-docs` · `apidocsmith` · `cli` · `mern-packages` · `merndev` · `mobile` · `nodejs` · `npm-pm` · `observability` · `openapi` · `react` · `react-native` · `swagger` · `typescript`

**React Native** library.

Generate beautiful API documentation automatically.

## Features

- Swagger generation
- request examples
- response examples
- auth documentation
- route grouping
- API versioning

## Example

```ts
import { apidocsmith } from "@mr-aftab-ahmad-khan/apidocsmith";

apidocsmith.scan(app);
```

## Why it matters

API documentation is usually outdated or missing completely.
